## Development Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Chrome/Firefox browser

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run on dev mode: `npm run dev`
4. Create build (in dist): `npm run build`
